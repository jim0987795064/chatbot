# README.md

## Prerequisites

OS: Ubuntu 20.04 (WSL)
Python: 3.8.5

```bash
# Create python virtual environment
python3 -m venv <venv-name>
# Activate virtual environment
source venv/bin/activate
# Install requirements
cat requirements.txt | xargs -n 1 pip install
```
